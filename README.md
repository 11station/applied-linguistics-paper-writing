# Applied Linguistics Paper Writing

A Codex skill for applied linguistics, second language acquisition (SLA), TESOL, EFL/ESL, and English academic writing.

This skill includes a built-in 2026-oriented reference list of linguistics and applied linguistics SSCI Q1 journals, together with journal-selection guidance for SLA, TESOL, EFL/ESL, L2 writing, language assessment, corpus-based studies, discourse/pragmatics, and AI-assisted language teaching research. Users should still verify each journal's current scope, ranking, impact factor, and author guidelines before submission.

It helps researchers and graduate students work through applied linguistics papers with a focus on research questions, literature reviews, theoretical framing, methodology, data interpretation, English academic prose, reviewer simulation, journal submission preparation, and AI-style revision.

## What This Skill Supports

- Topic refinement and research question design
- Literature review planning and synthesis
- Theoretical framework development
- Methodology and analysis-plan writing
- Results and discussion drafting
- English academic polishing for applied linguistics papers
- AI-style writing detection and revision
- SSCI/TESOL-oriented submission preparation
- Reviewer-style feedback before submission
- Citation, data, and statistical-result verification workflows

## Main Research Areas

- Second language acquisition and instructed SLA
- TESOL, EFL, ESL, and language pedagogy
- L2 writing and academic writing
- Language assessment and feedback
- Corpus-based applied linguistics
- Discourse, pragmatics, and classroom interaction
- AI-assisted language teaching and learning

## Important Boundaries

This skill does not fabricate data, participant information, interview excerpts, classroom observations, statistical results, references, DOIs, reviewer comments, or journal information.

It is designed to support academic writing and research thinking, not to replace real data analysis, ethics review, supervisor feedback, or journal-specific author guidelines.

## Installation

Clone this repository into your Codex skills directory:

```bash
git clone https://github.com/11station/applied-linguistics-paper-writing.git ~/.codex/skills/applied-linguistics-paper-writing
```

Restart Codex after installation so the skill can be discovered.


```text
Use the applied-linguistics-paper-writing skill to revise this SLA literature review in a more publishable academic style.
```

```text
Use the applied-linguistics-paper-writing skill to simulate reviewer comments for my TESOL manuscript.
```

# 应用语言学论文写作

这是一个用于 Codex 的应用语言学论文写作 skill，面向应用语言学、二语习得（SLA）、TESOL、EFL/ESL 和英文学术写作。

这个 skill 内嵌了面向 2026 年的语言学与应用语言学 SSCI Q1 期刊参考清单，并提供二语习得、TESOL、EFL/ESL、二语写作、语言测评、语料库研究、话语/语用研究和 AI 辅助语言教学等方向的选刊建议。投稿前仍需核对目标期刊官网上的最新 scope、分区、影响因子和 author guidelines。

它可以帮助研究者和研究生处理应用语言学论文写作中的多个环节，包括研究问题设计、文献综述、理论框架、研究方法、数据解释、英文学术表达、模拟审稿、期刊投稿准备，以及去 AI 味修改。

## 支持内容

- 选题聚焦与研究问题设计
- 文献综述规划与综合
- 理论框架搭建
- 方法论与分析计划写作
- 结果与讨论部分起草
- 应用语言学论文英文润色
- AI 写作痕迹识别与改写
- SSCI/TESOL 类期刊投稿准备
- 投稿前模拟审稿反馈
- 引用、数据和统计结果核实流程

## 重点研究方向

- 二语习得与课堂二语习得
- TESOL、EFL、ESL 与语言教学
- 二语写作与学术写作
- 语言测评与反馈研究
- 语料库应用语言学
- 话语、语用与课堂互动
- AI 辅助语言教学与学习

## 重要边界

这个 skill 不会编造数据、参与者信息、访谈摘录、课堂观察、统计结果、参考文献、DOI、审稿意见或期刊信息。

它的作用是辅助学术写作和研究思考，而不是替代真实数据分析、伦理审查、导师反馈或目标期刊的投稿指南。
